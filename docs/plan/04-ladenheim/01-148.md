# 04-ladenheim / 01 — Reproduce structural 148-network Ladenheim set

Status: `todo`

## Goal

Reproduce the 148 primitive RLC networks in the historical starting scope:

```text
R + L + C <= 5
L + C <= 2
```

The 148 are the structural catalogue after colour-preserving graph
2-isomorphism and exclusion of trivially same-kind series/parallel-reducible
candidates. They are not a raw pre-canonical candidate count. This starting set
includes eight networks with four resistors and one reactive element; those are
removed later when forming the canonical 108-network catalogue.

## Tasks

- Define the primitive candidate representation with individual R/L/C component
  edges.
- Define the colour-preserving, two-terminal 2-isomorphism contract, including
  terminal reversal, internal relabelling, deformation, separation, and series
  interchange.
- Decide whether a source-augmented graph is the cleanest way to preserve the
  port in the 2-isomorphism signature.
- Encode the historical same-kind series/parallel rejection rule used before the
  148 catalogue is formed.
- Produce descriptors and repository representatives for the 148 entries.

## Done means

- The 148 representatives are generated from the documented scope and contract.
- Each representative has a stable descriptor and provenance record.
- The validation path distinguishes graph isomorphism, 2-isomorphism, RICE local
  series/parallel reduction, and later electrical transformations.
